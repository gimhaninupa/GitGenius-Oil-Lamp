# 🌟 GitGenius Oil Lamp

A virtual oil lamp ceremony application designed specifically for premium events. It features a dual-screen setup, allowing you to walk around with a tablet (acting as the controller) to light the virtual oil lamp displayed on a separate large screen (projector, TV, or stage screen) in real time.

🌐 **Live Demo (Lamp):** [https://gitgenius-oil-lamp.onrender.com/](https://gitgenius-oil-lamp.onrender.com/)
📱 **Live Demo (Controller):** [https://gitgenius-oil-lamp.onrender.com/controller.html](https://gitgenius-oil-lamp.onrender.com/controller.html)

> **⚠️ Important:** Always reset the controller and refresh the lamp page before testing the lamp.

---

## ✨ Features

* **Dual-Screen Sync:** Real-time WebSockets communication between a tablet controller and a large display screen.
* **📱 Tablet Controller:** Interactive interface with guest lineup management (add, remove, drag-and-drop reorder) and a glowing touch flame button.
* **📺 Large Screen Display:** Visual oil lamp altar, custom candle wicks, ambient particle flames, and a final climax logo reveal.
* **🎵 Sound Synthesis:** Dynamic chime frequencies for each wick lit and a major chord synthesizer swell for the finale.
* **⚡ Auto-Sync & Fallback:** Instantaneous standard WebSockets action with offline `localStorage` fallback mechanism.
* **✨ Premium Animations:** Glowing Bezier-arc sparks, particle bursts, floating names, and premium announcement banners.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend Framework:** Node.js
* **Communication:** Native WebSockets
* **Deployment:** Render
